# Dashboard vs Framework - Quick Reference

## TL;DR

**Framework (this repo):** Digital company for solo founders. Works standalone.

**Dashboard (dashboard/ subfolder):** Separate project for managing multiple AI companies. Optional.

## Framework (Your Focus)

### What It Is
A digital enterprise that works FOR you through AI assistants.

### How It Works

## Dashboard (Different Project)

### What It Is
A web app for managing MULTIPLE agent companies at scale.

### How It Works
```

```

### What You Need

### Status
